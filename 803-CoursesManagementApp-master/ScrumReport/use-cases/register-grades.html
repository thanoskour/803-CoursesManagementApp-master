Register Grades
Data:
  • <grades id>
  • <project grade>
  • <exam grade>
  • <semester>
Actors
  1. Professor
Pre Conditions
  1. Course object should exist
  2. Student list should have at least one item
Main Flow:
  1. Professor issues the "Add Grades" command
  2. Professor fills in the data textfields acordingly
  3. Professor presses the 'OK' button
  4. System refreshes the Student item view
Field not filled (1)
  1. System notifies professor that a text fields is not filled with data.
  2. System does not proceed and prompts professor to fill in the remaining fields (highlights with red)
Post Conditions
  1. Grades of student should get added for the current semester
