Update Course Descirption
Data:
  • <course id>
  • <course name>
  • <syllabus>
  • <year>
  • <semester>
Actors
  1. Professor
Pre Conditions
  1. Course item we want to update should exist
Main Flow:
  1. Professor issues "Update" command
  2. System prompts professor with form
  3. Professor fills in the data textfields acordingly
  4. Professor presses the 'OK' button
  5. System repeats the 'Press Browse Button' use case
Post Conditions
  1. List of courses is updated with the new course data
